#Bomber Game

## Requirements

npm: v3.10.10

node: v6.11.0

gulp: CLI version 3.9.1

mongodb: v3.4.2

## Installation

Create "bomber" database in mongodb;

```gherkin
npm install;
cd app;
bower install;
gulp;
cd ..
npm start;
```
## View in browser
```gherkin
http://localhost:3000/
```
