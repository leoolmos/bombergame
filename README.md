#Bomber Game

## Requirements

npm: v3.10.10

node: v6.11.0

gulp: CLI version 3.9.1

mongodb: v3.4.2

## Installation

Make sure mongo is running.

```gherkin
npm install;
cd app;
bower install;
gulp;
cd ..
npm start;
```
## View in browser
```gherkin
http://localhost:3000/
```

## How to
- In order to see your friends in the list, they also need to log into the application to create a gamer profile.
